Embedded **Custom 3D Teacup Canvas** using React Three Fiber:
- Dynamic mirror chrome material with physically-based rendering
- Controlled rotation matrix with smooth physics interpolation

- Dark mode by default with seamless light mode toggle

- Tilt effects with directional glow following cursor
- Responsive animations powered by Framer Motion

## 🛠️ Tech Stack

### **Frontend**
- **Framework:** Next.js 14+ (App Router)
- **UI/Styling:** React + Tailwind CSS (Fluid responsive, custom gradients)
- **Animations:** Framer Motion (smooth, performant)
- **3D Graphics:** React Three Fiber + `@react-three/drei` (WebGL)
- **Icons:** React Icons, Lucide React
- **Components:** Shadcn/ui (accessible primitives)

### **Backend & Data**
- **Language:** Python, TypeScript, JavaScript
- **ML/Data:** Scikit-Learn, XGBoost, TensorFlow, Pandas, Plotly
- **APIs:** REST (Requests, Axios), Streamlit dashboards
- **Databases:** SQL, File I/O, joblib serialization

### **Deployment**
- **Hosting:** Vercel (Next.js), Netlify (static sites), GitHub Pages
- **Package Manager:** npm, pip
- **Version Control:** Git + GitHub
